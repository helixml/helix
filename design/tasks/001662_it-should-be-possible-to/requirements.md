# Requirements: Show Task Priority on Kanban Card

## User Story

As a user viewing the Kanban board, I want to see the priority of each task on its card so I can understand why certain tasks appear higher in the backlog without having to open each task.

## Acceptance Criteria

1. Each task card on the Kanban board displays the task's priority level.
2. The priority indicator is subtle — small text or a minimal visual element, not a bold badge or exclamation mark.
3. The four priority levels are visually distinct: `critical`, `high`, `medium`, `low`.
4. Tasks with no priority set show nothing (or a neutral/default state).
5. The priority display does not significantly increase card height or visual clutter.
6. The indicator is consistent with the existing card design language (muted colors, small typography).
