Update ZED_COMMIT to latest zed-4 main after upstream merge

## Summary

Updates `sandbox-versions.txt` to pin `ZED_COMMIT` to the new zed-4 `main` HEAD after the upstream Zed merge PR is merged. This causes CI to build the updated Zed binary with all upstream changes included.
