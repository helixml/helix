# Requirements

## User Story

As a spec task agent, I need to know that I can use the Chrome MCP server with DuckDuckGo to search the web, so that I can research solutions, look up documentation, and find answers during planning and implementation.

## Acceptance Criteria

- [ ] The planning prompt template includes a note about using Chrome MCP with DuckDuckGo for web search
- [ ] The implementation prompt template includes the same note
- [ ] The note is concise and placed logically near the existing Chrome MCP / Visual Testing sections
